# D2C Skincare — Startup Financial Model

**Ground-up financial model for a Direct-to-Consumer skincare brand with ₹20L startup capital**

---

## Project Overview

This project presents a comprehensive financial model built from scratch for a D2C skincare startup. The model covers full business planning — from unit economics and marketing budget allocation to breakeven analysis and 3-year financial projections — designed to simulate real startup fundraising and operational planning scenarios.

---

## What's Inside

### 1. Unit Economics
- Revenue per order, AOV (Average Order Value)
- CAC by channel — paid social, influencer, organic
- LTV based on repeat purchase rate and retention curve
- LTV/CAC ratio and payback period
- Contribution margin per SKU

### 2. Financial Projections (3-Year)
- Month-by-month revenue and cost build
- Gross margin and EBITDA trajectory
- Working capital and inventory assumptions
- Cash burn and runway analysis

### 3. Budget Allocation
- ₹20L capital deployment plan
- Marketing vs inventory vs operations split
- Sensitivity analysis on CAC and conversion rate

### 4. Breakeven Analysis
- Fixed vs variable cost structure
- Breakeven units and revenue threshold
- Time-to-breakeven under base and bear scenarios

---

## Key Assumptions

| Parameter | Value |
|---|---|
| Startup Capital | ₹20,00,000 |
| AOV | ₹800 – ₹1,200 |
| Gross Margin Target | 60–65% |
| CAC (Paid) | ₹350 – ₹500 |
| Repeat Purchase Rate | 25–30% |
